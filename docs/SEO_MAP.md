# DFY SEO Map

This map defines the approved route strategy before page implementation. It should guide page metadata, headings, internal links, CTAs, and schema. Do not add fake testimonials, fake logos, fake metrics, fake awards, or unverified claims.

## Global SEO Rules

- Brand: Done For You / DFY
- Canonical base: `https://www.doneforyou.agency`
- Primary CTA target: `/free-audit`
- Secondary CTA target where relevant: `/book-a-call`
- Global schema: `Organization`, `WebSite`
- Inner pages should use `BreadcrumbList`
- Service pages should use `Service`
- FAQ sections should use `FAQPage`
- Blog and guide pages should use `Article` where the content is article-style
- Listing pages should use `CollectionPage`
