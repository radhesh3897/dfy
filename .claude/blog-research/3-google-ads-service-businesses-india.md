## Practical guide content
- Structure: theme-based ad groups (5–15 kw, not SKAGs); separate Branded/Competitor/Category; one theme→one offer→one LP.
- Match types "barbell": Exact+Phrase on proven BOFU/brand; Broad ONLY with Smart Bidding + tracking + tight negatives.
- Negatives — universal: jobs/salary, free/cheap, course/training, DIY, forums. INDIA must-blocks: naukri, internshala, indeed, justdial, sulekha, urbancompany, indiamart, magicbricks, 99acres, olx, quikr. [lenoretech.in](https://www.lenoretech.in/blog/google-ads-negative-keywords-checklist.html)
- BOFU 70–80% budget: "near me", "[service]+[city]", hire/book/cost/quote, best/top/agency/firm.
- Landing pages: dedicated per service (never homepage), above-fold form, minimal fields (7→3 lifts conv 200%+), message match, <3s load, click-to-call + WhatsApp button, mobile-first (70%+ India traffic).
- Conversion tracking: native tag + Enhanced Conversions + Offline Conversion Import / Enhanced Conversions for Leads feeding qualified-lead values back = biggest junk-lead lever. Deadlines: OCI/ECL → Data Manager API June 15 2026; Consent Mode v2 for DPDP.
- CALL TRACKING (key India section): use call assets on RSAs NOT call-only ads (deprecated: no new creation Feb 2026, impressions end Feb 2027). Google call reporting caveat: caller numbers often not captured >15s "except in India". CallRail does NOT serve India — use Exotel or Nimbata. Set 30–60s min-call quality filter. [support.exotel.com](https://support.exotel.com/support/solutions/articles/3000080891)

## Google vs Meta
Intent (Google captures demand) vs interruption (Meta creates demand). Pick Google for urgent/local services, appointment outcomes, limited budget needing immediate revenue (70%+ to Google for local). Meta for demand creation + retargeting. India CPL: Meta cheaper per lead almost everywhere, Google converts faster. Documented case: Meta 3x clicks but 0 qualified leads vs Google. Consensus: run both. [orynet.com](https://orynet.com/cost-per-lead-in-india-by-industry-google-meta-comparison-2026/)

## Common mistakes
No/broken conversion tracking (~40% of audited accounts); ignoring phone calls; no negatives; broad match no guardrails; high-volume over high-intent keywords; homepage traffic; weak/slow LP; Smart Bidding before ~30 conv/mo; set-and-forget; over-editing (resets learning); loose geo; not tracking lead quality.
